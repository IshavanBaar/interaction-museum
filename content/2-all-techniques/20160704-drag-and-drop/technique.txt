Title: Drag-and-Drop

----

Description: Drang-and-Drop is one of the most popular direct manipulation techniques. The user can select a virtual object by grabbing it and then dragging it to a different location or onto another virtual object. This can trigger a command linked to target object.  This technique is especially useful on touch-screen devices.
